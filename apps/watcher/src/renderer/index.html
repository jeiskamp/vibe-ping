    Find your Discord webhook URL

    1. Open Discord and go to the server and channel where you want VibePing updates to appear.
    2. Open Edit Channel, then Integrations, then Webhooks.
    3. Create a webhook or choose an existing one, then click Copy Webhook URL.
    4. Paste that URL into VibePing and use Test connection when you are ready.
